# Socialmedia
A Social media Website inspired from Instagram, where users can signup, login, post images and view other people posts.

#### Technology Used
1. Django 
2. Postgresql
3. AWS (Amazon web services) S3 Bucket for storing all images.
4. Heroku for hosting
